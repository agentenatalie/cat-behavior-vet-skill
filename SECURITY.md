# Security and Safety Policy

## Supported Versions

This repository currently supports the `main` branch.

## Sensitive Data

Do not open public issues or pull requests that include:

- API keys or tokens
- `.env` files
- Zotero local library exports, notes, annotations, or attachments
- Private case details that identify a person, household, clinic, or animal
- PDFs or generated full-text/abstract corpus files unless redistribution rights are clear

If you accidentally expose a secret, revoke it first, then open an issue describing the affected file path without pasting the secret.

## Medical Safety

This project is not a diagnosis or treatment service. It is an evidence-retrieval and reasoning aid for AI runtimes.

It does not diagnose, prescribe, determine drug dose, adjust medication, manage emergencies, recommend delaying care, or create a veterinarian-client-patient relationship (VCPR).

Urgent or high-risk situations should be handled by local in-person care:

- Human or animal injury
- Escalating aggression
- Sudden behavior change
- Suspected pain or illness
- Welfare risk
- Medication questions requiring prescription decisions
- Active human-safety threats

## Reporting a Security Issue

Open a GitHub issue with a short description and reproduction steps. Do not include secrets, private case data, or copyrighted paper text.
