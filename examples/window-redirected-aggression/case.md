# Window-Triggered Aggression

Fictional user prompt:

```text
Use $cat-behavior-vet to assess this case:

My 4-year-old neutered indoor cat saw an outdoor cat through the living-room
window. About 20 seconds later he attacked my lower leg. He bit and scratched;
one bite broke skin. He has done this twice this month, always near the same
window. He recovered after about 40 minutes. No children at home. There is one
other cat, currently separated. No recent vet exam.
```

Expected skill behavior:

1. Ask for missing medical and incident details.
2. Give immediate bite-wound safety guidance.
3. Confirm the case summary before a full consult.
4. Retrieve evidence on owner-directed aggression, redirected aggression,
   stress, and medical differentials.
5. Search similar public real-world cases if web access is available.
6. Produce a cited consult with safety, management, behavior modification, and
   escalation thresholds.
