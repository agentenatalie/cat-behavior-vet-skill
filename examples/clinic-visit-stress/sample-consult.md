# Sample Consult: Clinic Visit Stress Handling Flow

This is a fictional example showing the expected skill output structure. It is
not veterinary advice for a real animal.

## Confirmed Case Summary

- 8-year-old spayed indoor female cat.
- Hides when the carrier appears.
- Pants in the car and sometimes urinates in the carrier.
- Clinic suggested a pre-visit medication; user wants evidence-based handling
  and preparation steps first.
- Still needs confirmation: respiratory disease, heart disease, pain, prior
  medication reactions, and visit urgency.

## Core Judgment

This is a high-stress veterinary-visit response. The carrier, car, clinic odor,
and handling have become a connected trigger chain. The goal is to change
"carrier appears = immediate clinic trip" into "carrier is a routine, safe,
rewarding place" and to coordinate low-stress handling with the clinic. Whether
pre-visit medication is appropriate is a veterinarian decision.

## Step-by-Step Execution Flow

1. Decide whether the visit is medically urgent.
   - If there is respiratory distress, obvious pain, urinary blockage concern,
     severe weakness, or another red flag: call the veterinarian first and treat
     it as medical priority; do not wait for training.
   - If it is not urgent: start preparation.
2. Keep the carrier out at home with bedding, treats, and familiar scent.
3. Check whether the cat can approach the carrier and eat.
   - If not: reduce difficulty by moving the carrier farther away and feeding
     near it.
   - If yes: continue.
4. Train in small steps: enter carrier -> door briefly closes -> carrier is
   lifted -> carrier goes into parked car -> engine turns on -> short drive.
5. Watch at every step.
   - If panting, urination, freezing, escape attempts, or food refusal appears:
     go back one step and shorten the session.
   - If stable: log the response and continue at the next short session.
6. Contact the clinic in advance for a low-stress plan.
7. If training is insufficient or the visit must happen soon: discuss pre-visit
   medication with the veterinarian.

## Handling Plan

1. **Normalize the carrier**: keep it out permanently, not only before vet
   visits. Add familiar bedding, treats, and toys.
2. **Train in small steps**: approach carrier -> enter -> brief door closure ->
   lift -> parked car with engine off -> engine on -> short drive.
3. **Stay below threshold**: continue only while the cat can eat, disengage, and
   keep a non-rigid body.
4. **Coordinate with the clinic**: schedule a quiet time, cover the carrier,
   reduce dog exposure, use gentle handling, and pause if distress escalates.
5. **Medication is a veterinary discussion**: if the visit is necessary before
   training is sufficient, or the cat already has severe panic signs, discuss
   pre-visit medication with the veterinarian.
6. **Avoid common mistakes**: do not force the cat out, punish, flood, or repeat
   long car rides while the cat is panicking.
