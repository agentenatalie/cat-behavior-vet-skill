# 就诊压力

虚构用户输入：

```text
Use $cat-behavior-consult to assess this case:

我家 8 岁已绝育室内母猫坐车和去兽医诊所时会非常恐慌。猫包一出现她就躲起来，在车里会喘，有时还会尿在猫包里。诊所建议下次就诊前用药，但我想先知道有哪些循证的处理和准备步骤。
```

预期 skill 行为：

1. 追问医学状态、就诊紧急程度、呼吸症状、既往用药反应、猫包历史和诊所 handling。
2. 区分急性医学风险和行为准备问题。
3. 检索 feline stress、handling 和 pre-appointment medication 相关证据。
4. 给出猫包训练、运输设置、诊所沟通，以及什么时候需要和兽医讨论用药的方案。
